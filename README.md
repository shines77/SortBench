# SortBench

## 介绍

常用排序算法的基准测试。

比较类排序：

+ 插入排序
+ 希尔排序
+ 选择排序
+ 冒泡排序
+ 堆排序
+ 归并排序
+ Timsort
+ 快速排序
+ [Pattern-defeating QuickSort](https://github.com/orlp/pdqsort)

非比较排序：

+ 计数排序
+ 基数排序
+ 桶排序
+ [RoaringBitmap Sort](https://gitee.com/shines77/rbsort)
